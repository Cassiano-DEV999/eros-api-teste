# **App Name**: Mom's Companion

## Core Features:

- Appointment Reminders: Allow pregnant individuals to set reminders for appointments (events, location, day, time). The app will generate notifications on their mobile device as the appointment nears.
- Medication and Supplement Tracker: Enable users to log medications and supplements, including name, schedule, and treatment duration. The app sends timely notifications. Will generate a warning message to the user if there are potential adverse drug interactions, making it an AI tool.
- Healthcare Locator: Displays a map with nearby UBS (Unidades Básicas de Saúde) and ESF (Estratégia Saúde da Família) locations within Sorocaba and the surrounding region.
- Teleconsultation Booking & Payment: Allows users to schedule and pay for teleconsultations with clinical pharmacists via PIX (QR code generation) or credit card.
- Educational Resources: Provides access to informative PDF documents on various topics related to pregnancy, such as supplements, vitamins, vaccines, prenatal care, STIs, postpartum, breastfeeding, etc.
- Support Network Access: Generates a unique access code (like the Flo app) that users can share with their support network, granting them access to the user's appointment schedule, treatment information, and healthcare locations.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke feelings of calmness and nurturing.
- Background color: Light gray (#F0F0F0) to ensure a clean, uncluttered backdrop.
- Accent color: Pale rose (#FFC0CB) to add a gentle touch of warmth.
- Body and headline font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel.
- Use gentle, rounded icons that feel welcoming and approachable.
- A clean, spacious layout that is intuitive and easy to navigate. Each section should be clearly delineated.
- Subtle, gentle animations, such as a soft fade-in, when content loads to create a smooth and engaging experience.
